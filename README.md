# 🚀 RAG Assistant (GigaChat + ChromaDB)

## 📌 О проекте

Интерактивный AI-ассистент на основе **RAG (Retrieval-Augmented Generation)**, который отвечает на вопросы пользователя, используя собственную базу знаний.

Проект демонстрирует построение полноценной RAG-системы с:

* векторной базой данных
* кешированием ответов
* логированием
* интерактивным режимом общения

---

## ⚙️ Как это работает

1. Пользователь задаёт вопрос
2. Система ищет релевантные документы в ChromaDB
3. Передаёт контекст в LLM (GigaChat)
4. Генерирует ответ
5. Сохраняет результат в кеш

👉 При повторном запросе ответ берётся из кеша (ускорение + экономия ресурсов)

---

## 🧩 Архитектура

* **RAGAssistant** — генерация ответов
* **EmbeddingStore (ChromaDB)** — хранение и поиск документов
* **ResponseCache** — кеширование
* **Logging (logging)** — логирование работы системы
* **Interactive Mode** — CLI-интерфейс

## 🖥️ Запуск проекта

### 1. Клонировать репозиторий

```bash
git clone https://github.com/your-username/rag-assistant.git
cd rag-assistant
```

---

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

---

### 3. Настроить переменные окружения

Создай `.env` файл:

```env
GIGACHAT_CREDENTIALS=your_api_key_here
```

---

### 4. Запустить

```bash
python main.py
```

---

## 💬 Режимы работы

### 1. Интерактивный режим

* ввод вопросов вручную
* команды:

  * `cache` — размер кеша
  * `clear_cache` — очистка кеша
  * `stats` — статистика

### 2. Демо-режим

* заранее подготовленные вопросы
* демонстрация работы кеша

---

## 📂 Логирование

Все действия записываются:

* в консоль
* в файл `app.log`

Пример:

```
2026-04-15 12:00:01 | INFO | User input: Что такое RAG?
2026-04-15 12:00:03 | INFO | Ответ сгенерирован за 2.1 сек
```
